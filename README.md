# frontend_ToDo_app
## The famous ToDo App!

## This is the famous ToDo app build wtih:
* Advanced CSS and HTML
* Javascript
* Body-parser
* Nodejs
* Express
* EJS templates


## Functionalities:
* Add a todo item
* Delete (strike-through) an item
* Currently only suports up to 5 items
